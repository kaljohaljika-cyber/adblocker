Release note

Version 1.1.3 reduces the signed APK download from about 101 MB to about 13.3 MB by removing native debug symbols and compressing the OpenVPN libraries. It also stages future updates in private non-purgeable storage so Android cache cleanup cannot remove a download in progress. Automatic signed updates, trusted-device profile protection, and fail-closed App Tunnel behavior are unchanged. Version-code-11 users still need one manual matching-signature bridge install to version 12 or later. VPN-minute balances are unchanged.
